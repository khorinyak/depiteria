# Depiteria — сайт (v4 — Ultra Minimal)

## Что внутри
- `index.html` — главная (лендинг)
- `privacy.html` — политика конфиденциальности (базовая версия)
- `assets/` — изображения (jpg/png) + оптимизированные webp

## Как открыть
Откройте `index.html` в браузере (это статический лендинг).

## Быстрая публикация
**Netlify:** Add new site → Deploy manually → перетащите ZIP или папку.  
**GitHub Pages:** загрузите файлы в репозиторий → Settings → Pages.

## Где править текст
Откройте `index.html` и найдите блоки:
- `Гладкость без переплат` (первый экран)
- `Спец‑предложение` (промокод)
- `Филиалы` (список адресов/ссылок)
